# 📊 Data Analysis: SQL + Power BI

End-to-end data analytics project involving data cleaning, transformation, and visualization to derive meaningful business insights.

## 🛠 Tools & Technologies

* **SQL**: Data extraction, cleaning, and query optimization
* **Power BI**: Interactive dashboard creation
* **DAX**: KPI calculations and data modeling

## 📊 Key Features

* Customer segmentation based on age, gender, and account type
* Analysis of transaction trends on monthly and yearly basis
* KPI tracking including:

  * Total Balance
  * Average Transactions
  * Overall customer activity insights

## 📈 Dashboard

An interactive Power BI dashboard designed to analyze customer behavior and transaction patterns, enabling better understanding of trends and performance metrics.
